# Fibre Laser Toolkit Project

A Python simulation toolkit for fibre optics and laser modelling.

## Features (in development)
- Step-index fibre mode solver (LP + vectorial HE/EH modes)
- Hollow-core fibre modelling
- Bent fibre analysis via conformal mapping
- Beam shaping via mode superposition
- Full fibre laser modelling (CW, pulsed, GNLSE)

## Setup
Install uv then run:
    uv sync
